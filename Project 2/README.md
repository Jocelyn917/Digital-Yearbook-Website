# Digital Yearbook

A full-stack-style React + Vite prototype that uses localStorage for persistence, so it runs without a backend.

## Run it

```bash
npm install
npm run dev
```

## Included

- Digital classmate profiles with photos, bios, memories, notes, and mailto links
- Searchable/filterable yearbook directory
- Profile detail modal with locally saved private notes
- Hangout scheduler with invited classmates and upcoming list
- Anniversary/reunion reminders with edit and delete flows
- Seed sample classmates, hangouts, and reminders
